# 🏥 General Surgeon Healthcare Website

A modern and responsive healthcare website template designed for **General Surgeons, Hospitals, Clinics, and Healthcare Professionals**.

Built with clean HTML, CSS, and Vanilla JavaScript, this project offers a professional online presence for medical practices with appointment booking and patient-friendly design.
